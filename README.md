# Лабораторная работа №1: Сравнение методов решения СЛАУ

## 1. Постановка задачи
Целью работы является реализация и сравнительный анализ алгоритмов решения систем линейных алгебраических уравнений (СЛАУ):
1. Метод Гаусса без выбора ведущего элемента.
2. Метод Гаусса с выбором главного элемента по столбцу (частичный выбор).
3. Метод LU-разложения.

Основная задача — продемонстрировать преимущество LU-разложения при решении серий систем с одной и той же матрицей, но разными правыми частями, а также оценить влияние выбора ведущего элемента на точность вычислений на плохо обусловленных матрицах (матрица Гильберта).

## 2. Описание алгоритмов

### Метод Гаусса
Алгоритм приводит расширенную матрицу системы к ступенчатому виду (треугольному) методом последовательного исключения переменных.
- **Без выбора ведущего элемента:** На каждом шаге деление производится на диагональный элемент $A_{kk}$. Если $A_{kk} \approx 0$, метод дает большую погрешность или завершается ошибкой.
- **С частичным выбором:** Перед исключением на $k$-м шаге строки меняются местами так, чтобы на диагонали оказался элемент с максимальным модулем в текущем столбце. Это повышает численную устойчивость.

### LU-разложение
Матрица $A$ представляется в виде произведения нижней треугольной матрицы $L$ (с единицами на диагонали) и верхней треугольной матрицы $U$:
$$ A = L \cdot U $$
Решение системы $Ax=b$ сводится к двум этап:
1. Решение $Ly = b$ (прямая подстановка).
2. Решение $Ux = y$ (обратная подстановка).

*Примечание:* В данной реализации LU-разложение выполняется без перестановок строк (в соответствии с требованиями варианта).
